:::caution Before proceeding with the upgrade
Please read the **whole guide** in order to have a good understanding of how all the components fit together.

You need to be careful not to tear or damange some of the cables when handling the casing of the device.
:::

:::success
**Prerequesites** for this guide:
- An [AutoPi](https://www.autopi.io) [TMU CM4](https://www.autopi.io/hardware/autopi-tmu-cm4) device
- Heatsink package provided by [AutoPi](https://www.autopi.io)
:::

### Step-by-Step Guide

<SideBySide>
	<p>
		<strong>Step 1: </strong>
		Open up the bag with the heatsink and the accessories.
	</p>
	<img alt="Step 1" src={useBaseUrl('/img/hardware/autopi_tmu_cm4/installing_heatsink/heatsink1.jpg')}/>
</SideBySide>
<br/>

<SideBySide>
	<p>
		<strong>Step 2: </strong>
		Find the 4 metal screws and the 4 spacers. 
	</p>
	<img alt="Step 2" src={useBaseUrl('/img/hardware/autopi_tmu_cm4/installing_heatsink/heatsink2.jpg')}/>
</SideBySide>
<br/>

<SideBySide>
	<p>
		<strong>Step 3: </strong>
		Apply the thermal tape to the Ethernet IC and the CPU as show on the image. 
	</p>
	<img alt="Step 3" src={useBaseUrl('/img/hardware/autopi_tmu_cm4/installing_heatsink/heatsink3.jpg')}/>
</SideBySide>
<br/>

<SideBySide>
	<p>
		<strong>Step 4: </strong>
		Fix the heatsink on the thermal tape and fasten it using the metal screw. Remember to use the spacer between the screw and the heatsink. 
	</p>
	<img alt="Step 4" src={useBaseUrl('/img/hardware/autopi_tmu_cm4/installing_heatsink/heatsink4.jpg')}/>
</SideBySide>
<br/>

### Disscussion
<p>
	You have now successfully installed a heatsink onto your [AutoPi](https://www.autopi.io) [TMU CM4](https://www.autopi.io/hardware/autopi-tmu-cm4) device.
</p>
