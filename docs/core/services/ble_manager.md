## Handlers
### `context`

Gets current context.


----
### `flash_firmware`

Flashes new firmware to the BlueNRG device.


----
### `interface`

Manages the interface to the BlueNRG device.


----
### `mode`

Manages the low-level modes of the BlueNRG device.


----
### `query`

Queries a given BlueNRG ACI command.
