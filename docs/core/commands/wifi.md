## `wifi.help`

Shows this help information.


----
## `wifi.scan`

Give the list of Access Points and Ad-Hoc cells in range.

**OPTIONAL ARGUMENTS**

  - **`interface`** (str): Default is `wlan0`.


----
## `wifi.status`

Get current WPA/EAPOL/EAP status.

**OPTIONAL ARGUMENTS**

  - **`interface`** (str): Default is `wlan0`.
