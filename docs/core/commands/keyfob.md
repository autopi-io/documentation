## `keyfob.action`

Performs a key fob button action.

**ARGUMENTS**

  - **`*name`** (str): Name(s) of the action(s) to perform.


----
## `keyfob.context`

:::warning
**DEPRECATED**
Use 'manage context' instead.
:::
Gets current context.


----
## `keyfob.help`

Shows this help information.


----
## `keyfob.power`

Powers on/off key fob.

**OPTIONAL ARGUMENTS**

  - **`value`** (bool): Power on or off.
