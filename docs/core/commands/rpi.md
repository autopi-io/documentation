## `rpi.boot_time`

Get timestamp for last boot of system.


----
## `rpi.help`

Shows this help information.


----
## `rpi.hw_serial`

Get hardware serial.


----
## `rpi.temp`

Current temperature readings.


----
## `rpi.temp_cpu`

Current temperature of the ARM CPU.


----
## `rpi.temp_gpu`

Current temperature of the GPU.
